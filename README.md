# ①課題番号-プロダクト名

Talemuse（AI自叙伝）卒業制作

## ②課題内容（どんな作品か）

- Talemuseは物語を噛み締めるという意味の造語です。
AIを使い、低コストで誰もが簡単に自分の人生の軌跡を残せるサービスとして人々の記録を残していきたいです。
自分だけでなく、家族や周りの人の自叙伝が見れるようになると新しいコミュニケーションも生まれると思います。
より身近だからこそ知り得なかったことを知り、新しい絆を作っていきたいです。
